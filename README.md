# MiniGomoku
這是 NCU「計算機網路 Computer Network(CE3007A)」一課的 Socket Programming 作業。這個專案使用 Python、PyQt6、Pygame 實現了一個簡單的五子棋遊戲。

可透過 requirements.txt 安裝所有需要的函式庫：
``` bash
pip install -r requirements.txt
```
